# Rock-Paper-Scissors Game
A simple Rock-Paper-Scissors game built using Python, where the user plays against the computer.
The game follows standard rules, displays results clearly, and keeps track of scores across multiple rounds.

## Game Rules
- Rock beats Scissors
- Scissors beats Paper
- Paper beats Rock
- Same choice results in a tie
